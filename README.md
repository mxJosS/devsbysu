## Project Description

**DevsBySu** is a web application built with the Laravel framework. This project has been developed based on Clean Code principles to provide a great experience in both data administration and user interface.

### Main Features

* **MVC Architecture:** Solid implementation following Laravel design patterns.
* **Modern Interface:** Styled with Tailwind CSS and Flowbite for a responsive and minimalist design.
* **Data Management:** Utilizes Eloquent ORM for efficient queries and structured migrations.
* **Security:** Integrated authentication and route protection.

## Installation and Configuration
Follow these steps to replicate the development environment:
1. **Clone Repository**
   ```bash
   git clone [https://github.com/mxJosS/devsbysu.git](https://github.com/mxJosS/devsbysu.git)
   ``` 
2. Create the MySQL database conection and execute:
   ```bash
   php artisan migrate
   ```
3. Download the composer dependencies:
   ```bash
   composer install
   ```
